  59.  59 <h1>CAPTCHA Helper</h1>
  60.  60 
  61.  61 <p>The CAPTCHA Helper file contains functions that assist in creating CAPTCHA images.</p>
  62.  62 
  63.  63 
  64.  64 <h2>Loading this Helper</h2>
  65.  65 
  66.  66 <p>This helper is loaded using the following code:</p>
  67.  67 <code>$this->load->helper('captcha');</code>
  68.  68 
  69.  69 <p>The following functions are available:</p>
  70.  70 
  71.  71 <h2>create_captcha(<var>$data</var>)</h2>
  72.  72 
  73.  73 <p>Takes an array of information to generate the CAPTCHA as input and creates the image to your specifications, returning an array of associative data about the image.</p>
  74.  74 
  75.  75 <code>[array]<br />
  76.  76 (<br />
  77.  77 &nbsp;&nbsp;'image' => IMAGE TAG<br />
  78.  78 &nbsp;&nbsp;'time' => TIMESTAMP (in microtime)<br />
  79.  79 &nbsp;&nbsp;'word' => CAPTCHA WORD<br />
  80.  80 )</code>
  81.  81 
  82.  82  <p>The "image" is the actual image tag:
  83.  83 <code>&lt;img src=&quot;http://example.com/captcha/12345.jpg&quot; width=&quot;140&quot; height=&quot;50&quot; /&gt;</code></p>
  84.  84 
  85.  85  <p>The "time" is the micro timestamp used as the image name without the file
  86.  86  extension.  It will be a number like this:  1139612155.3422</p>
  87.  87 
  88.  88  <p>The "word" is the word that appears in the captcha image, which if not
  89.  89  supplied to the function, will be a random string.</p>
  90.  90 
  91.  91  <h3>Using the CAPTCHA helper</h3>
  92.  92 
  93.  93  <p>Once loaded you can generate a captcha like this:</p>
  94.  94 
  95.  95 <code>$vals = array(<br />
  96.  96 &nbsp;&nbsp;&nbsp;&nbsp;'word'  => 'Random word',<br />
  97.  97 &nbsp;&nbsp;&nbsp;&nbsp;'img_path'  => './captcha/',<br />
  98.  98 &nbsp;&nbsp;&nbsp;&nbsp;'img_url'  => 'http://example.com/captcha/',<br />
  99.  99 &nbsp;&nbsp;&nbsp;&nbsp;'font_path'  => './path/to/fonts/texb.ttf',<br />
  100. 100 &nbsp;&nbsp;&nbsp;&nbsp;'img_width'  => '150',<br />
  101. 101 &nbsp;&nbsp;&nbsp;&nbsp;'img_height' => 30,<br />
  102. 102 &nbsp;&nbsp;&nbsp;&nbsp;'expiration' => 7200<br />
  103. 103 &nbsp;&nbsp;&nbsp;&nbsp;);<br />
  104. 104 <br />
  105. 105 $cap = create_captcha($vals);<br />
  106. 106 echo $cap['image'];</code>
  107. 107 
  108. 108  <ul>
  109. 109  <li>The captcha function requires the GD image library.</li>
  110. 110  <li>Only the img_path and img_url are required.</li>
  111. 111  <li>If a "word" is not supplied, the function will generate a random
  112. 112  ASCII string.  You might put together your own word library that
  113. 113  you can draw randomly from.</li>
  114. 114  <li>If you do not specify a path to a TRUE TYPE font, the native ugly GD
  115. 115  font will be used.</li>
  116. 116  <li>The "captcha" folder must be writable (666, or 777)</li>
  117. 117  <li>The "expiration" (in seconds) signifies how long an image will
  118. 118  remain in the captcha folder before it will be deleted.  The default
  119. 119  is two hours.</li>
  120. 120  </ul>
  121. 121 
  122. 122  <h3>Adding a Database</h3>
  123. 123 
  124. 124  <p>In order for the captcha function to prevent someone from submitting, you will need
  125. 125  to add the information returned from <kbd>create_captcha()</kbd> function to your database.
  126. 126  Then, when the data from the form is submitted by the user you will need to verify
  127. 127  that the data exists in the database and has not expired.</p>
  128. 128 
  129. 129  <p>Here is a table prototype:</p>
  130. 130 
  131. 131 <code>CREATE TABLE captcha (<br />
  132. 132 &nbsp;captcha_id bigint(13) unsigned NOT NULL auto_increment,<br />
  133. 133 &nbsp;captcha_time int(10) unsigned NOT NULL,<br />
  134. 134 &nbsp;ip_address varchar(16) default '0' NOT NULL,<br />
  135. 135 &nbsp;word varchar(20) NOT NULL,<br />
  136. 136 &nbsp;PRIMARY KEY `captcha_id` (`captcha_id`),<br />
  137. 137 &nbsp;KEY `word` (`word`)<br />
  138. 138 );</code>
  139. 139 
  140. 140  <p>Here is an example of usage with a database.  On the page where the CAPTCHA will be shown you'll have something like this:</p>
  141. 141 
  142. 142 <code>$this-&gt;load-&gt;helper(&#x27;captcha&#x27;);<br />
  143. 143 $vals = array(<br />
  144. 144 &nbsp;&nbsp;&nbsp;&nbsp;&#x27;img_path&#x27;  =&gt; &#x27;./captcha/&#x27;,<br />
  145. 145 &nbsp;&nbsp;&nbsp;&nbsp;&#x27;img_url&#x27;  =&gt; &#x27;http://example.com/captcha/&#x27;<br />
  146. 146 &nbsp;&nbsp;&nbsp;&nbsp;);<br />
  147. 147 <br />
  148. 148 $cap = create_captcha($vals);<br />
  149. 149 <br />
  150. 150 $data = array(<br />
  151. 151 &nbsp;&nbsp;&nbsp;&nbsp;&#x27;captcha_time&#x27; =&gt; $cap[&#x27;time&#x27;],<br />
  152. 152 &nbsp;&nbsp;&nbsp;&nbsp;&#x27;ip_address&#x27; =&gt; $this-&gt;input-&gt;ip_address(),<br />
  153. 153 &nbsp;&nbsp;&nbsp;&nbsp;&#x27;word&#x27; =&gt; $cap[&#x27;word&#x27;]<br />
  154. 154 &nbsp;&nbsp;&nbsp;&nbsp;);<br />
  155. 155 <br />
  156. 156 $query = $this-&gt;db-&gt;insert_string(&#x27;captcha&#x27;, $data);<br />
  157. 157 $this-&gt;db-&gt;query($query);<br />
  158. 158 <br />
  159. 159 echo &#x27;Submit the word you see below:&#x27;;<br />
  160. 160 echo $cap[&#x27;image&#x27;];<br />
  161. 161 echo &#x27;&lt;input type=&quot;text&quot; name=&quot;captcha&quot; value=&quot;&quot; /&gt;&#x27;;</code>
  162. 162 
  163. 163  <p>Then, on the page that accepts the submission you'll have something like this:</p>
  164. 164 
  165. 165 <code>// First, delete old captchas<br />
  166. 166 $expiration = time()-7200; // Two hour limit<br />
  167. 167 $this-&gt;db-&gt;query(&quot;DELETE FROM captcha WHERE captcha_time &lt; &quot;.$expiration); <br />
  168. 168 <br />
  169. 169 // Then see if a captcha exists:<br />
  170. 170 $sql = &quot;SELECT COUNT(*) AS count FROM captcha WHERE word = ? AND ip_address = ? AND captcha_time &gt; ?&quot;;<br />
  171. 171 $binds = array($_POST[&#x27;captcha&#x27;], $this-&gt;input-&gt;ip_address(), $expiration);<br />
  172. 172 $query = $this-&gt;db-&gt;query($sql, $binds);<br />
  173. 173 $row = $query-&gt;row();<br />
  174. 174 <br />
  175. 175 if ($row-&gt;count == 0)<br />
  176. 176 {<br />
  177. 177 &nbsp;&nbsp;&nbsp;&nbsp;echo &quot;You must submit the word that appears in the image&quot;;<br />
  178. 178 }</code>
