  59.  59 <h1>Form Helper</h1>
  60.  60 
  61.  61 <p>The Form Helper file contains functions that assist in working with forms.</p>
  62.  62 
  63.  63 
  64.  64 <h2>Loading this Helper</h2>
  65.  65 
  66.  66 <p>This helper is loaded using the following code:</p>
  67.  67 <code>$this->load->helper('form');</code>
  68.  68 
  69.  69 <p>The following functions are available:</p>
  70.  70 
  71.  71 
  72.  72 
  73.  73 <h2>form_open()</h2>
  74.  74 
  75.  75 <p>Creates an opening form tag with a base URL <strong>built from your config preferences</strong>.  It will optionally let you
  76.  76 add form attributes and hidden input fields, and will always add the attribute <kbd>accept-charset</kbd> based on the charset value in your config file.</p>
  77.  77 
  78.  78 <p>The main benefit of using this tag rather than hard coding your own HTML is that it permits your site to be more portable
  79.  79 in the event your URLs ever change.</p>
  80.  80 
  81.  81 <p>Here's a simple example:</p>
  82.  82 
  83.  83 <code>echo form_open('email/send');</code>
  84.  84 
  85.  85 <p>The above example would create a form that points to your base URL plus the "email/send" URI segments, like this:</p>
  86.  86 
  87.  87 <code>&lt;form method="post" accept-charset="utf-8" action="http:/example.com/index.php/email/send" /></code>
  88.  88 
  89.  89 <h4>Adding Attributes</h4>
  90.  90 
  91.  91 <p>Attributes can be added by passing an associative array to the second parameter, like this:</p>
  92.  92 
  93.  93 <code>
  94.  94 $attributes = array('class' => 'email', 'id' => 'myform');<br />
  95.  95 <br />
  96.  96 echo form_open('email/send', $attributes);</code>
  97.  97 
  98.  98 <p>The above example would create a form similar to this:</p>
  99.  99 
  100. 100 <code>&lt;form method="post" accept-charset="utf-8" action="http:/example.com/index.php/email/send" &nbsp;class="email" &nbsp;id="myform" /></code>
  101. 101 
  102. 102 <h4>Adding Hidden Input Fields</h4>
  103. 103 
  104. 104 <p>Hidden fields can be added by passing an associative array to the third parameter, like this:</p>
  105. 105 
  106. 106 <code>
  107. 107 $hidden = array('username' => 'Joe', 'member_id' => '234');<br />
  108. 108 <br />
  109. 109 echo form_open('email/send', '', $hidden);</code>
  110. 110 
  111. 111 <p>The above example would create a form similar to this:</p>
  112. 112 
  113. 113 <code>&lt;form method="post" accept-charset="utf-8" action="http:/example.com/index.php/email/send"><br />
  114. 114 &lt;input type="hidden" name="username" value="Joe" /><br />
  115. 115 &lt;input type="hidden" name="member_id" value="234" /></code>
  116. 116 
  117. 117 
  118. 118 <h2>form_open_multipart()</h2>
  119. 119 
  120. 120 <p>This function is absolutely identical to the <dfn>form_open()</dfn> tag above except that it adds a multipart attribute,
  121. 121 which is necessary if you would like to use the form to upload files with.</p>
  122. 122 
  123. 123 <h2>form_hidden()</h2>
  124. 124 
  125. 125 <p>Lets you generate hidden input fields.  You can either submit a name/value string to create one field:</p>
  126. 126 
  127. 127 <code>form_hidden('username', 'johndoe');<br />
  128. 128 <br />
  129. 129 // Would produce:<br /><br />
  130. 130 &lt;input type="hidden" name="username" value="johndoe" /></code>
  131. 131 
  132. 132 <p>Or you can submit an associative array to create multiple fields:</p>
  133. 133 
  134. 134 <code>$data = array(<br />
  135. 135 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'name'&nbsp;&nbsp;=> 'John Doe',<br />
  136. 136 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'email' => 'john@example.com',<br />
  137. 137 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'url'&nbsp;&nbsp;&nbsp;=> 'http://example.com'<br />
  138. 138 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;);<br />
  139. 139 <br />
  140. 140 echo form_hidden($data);<br />
  141. 141 <br />
  142. 142 // Would produce:<br /><br />
  143. 143 &lt;input type="hidden" name="name" value="John Doe" /><br />
  144. 144 &lt;input type="hidden" name="email" value="john@example.com" /><br />
  145. 145 &lt;input type="hidden" name="url" value="http://example.com" /></code>
  146. 146 
  147. 147 
  148. 148 
  149. 149 
  150. 150 <h2>form_input()</h2>
  151. 151 
  152. 152 <p>Lets you generate a standard text input field.  You can minimally pass the field name and value in the first
  153. 153 and second parameter:</p>
  154. 154 
  155. 155 <code>echo form_input('username', 'johndoe');</code>
  156. 156 
  157. 157 <p>Or you can pass an associative array containing any data you wish your form to contain:</p>
  158. 158 
  159. 159 <code>$data = array(<br />
  160. 160 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'name'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> 'username',<br />
  161. 161 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'id'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> 'username',<br />
  162. 162 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'value'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> 'johndoe',<br />
  163. 163 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'maxlength'&nbsp;&nbsp;&nbsp;=> '100',<br />
  164. 164 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'size'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '50',<br />
  165. 165 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'style'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; => 'width:50%',<br />
  166. 166 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;);<br />
  167. 167 <br />
  168. 168 echo form_input($data);<br />
  169. 169 <br />
  170. 170 // Would produce:<br /><br />
  171. 171 &lt;input type="text" name="username" id="username" value="johndoe" maxlength="100" size="50" style="width:50%" /></code>
  172. 172 
  173. 173 <p>If you would like your form to contain some additional data, like Javascript, you can pass it as a string in the
  174. 174 third parameter:</p>
  175. 175 
  176. 176 <code>$js = 'onClick="some_function()"';<br />
  177. 177 <br />
  178. 178 echo form_input('username', 'johndoe', $js);</code>
  179. 179 
  180. 180 <h2>form_password()</h2>
  181. 181 
  182. 182 <p>This function is identical in all respects to the <dfn>form_input()</dfn> function above
  183. 183 except that is sets it as a "password" type.</p>
  184. 184 
  185. 185 <h2>form_upload()</h2>
  186. 186 
  187. 187 <p>This function is identical in all respects to the <dfn>form_input()</dfn> function above
  188. 188 except that is sets it as a "file" type, allowing it to be used to upload files.</p>
  189. 189 
  190. 190 <h2>form_textarea()</h2>
  191. 191 
  192. 192 <p>This function is identical in all respects to the <dfn>form_input()</dfn> function above
  193. 193 except that it generates a "textarea" type. Note: Instead of the "maxlength" and "size" attributes in the above
  194. 194 example, you will instead specify "rows" and "cols".</p>
  195. 195 
  196. 196 
  197. 197 <h2>form_dropdown()</h2>
  198. 198 
  199. 199 <p>Lets you create a standard drop-down field.  The first parameter will contain the name of the field,
  200. 200 the second parameter will contain an associative array of options, and the third parameter will contain the
  201. 201 value you wish to be selected.  You can also pass an array of multiple items through the third parameter, and CodeIgniter will create a multiple select for you. Example:</p>
  202. 202 
  203. 203 <code>$options = array(<br />
  204. 204 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'small'&nbsp;&nbsp;=> 'Small Shirt',<br />
  205. 205 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'med'&nbsp;&nbsp;&nbsp;&nbsp;=> 'Medium Shirt',<br />
  206. 206 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'large'&nbsp;&nbsp; => 'Large Shirt',<br />
  207. 207 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'xlarge' => 'Extra Large Shirt',<br />
  208. 208 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;);<br />
  209. 209 <br />
  210. 210 $shirts_on_sale = array('small', 'large');<br />
  211. 211 <br />
  212. 212 echo form_dropdown('shirts', $options, 'large');<br />
  213. 213 <br />
  214. 214 // Would produce:<br />
  215. 215 <br />
  216. 216 &lt;select name=&quot;shirts&quot;&gt;<br />
  217. 217 &lt;option value=&quot;small&quot;&gt;Small Shirt&lt;/option&gt;<br />
  218. 218 &lt;option value=&quot;med&quot;&gt;Medium  Shirt&lt;/option&gt;<br />
  219. 219 &lt;option value=&quot;large&quot; selected=&quot;selected&quot;&gt;Large Shirt&lt;/option&gt;<br />
  220. 220 &lt;option value=&quot;xlarge&quot;&gt;Extra Large Shirt&lt;/option&gt;<br />
  221. 221 &lt;/select&gt;<br />
  222. 222 <br />
  223. 223 echo form_dropdown('shirts', $options, $shirts_on_sale);<br />
  224. 224 <br />
  225. 225 // Would produce:<br />
  226. 226 <br />
  227. 227 &lt;select name=&quot;shirts&quot; multiple=&quot;multiple&quot;&gt;<br />
  228. 228 &lt;option value=&quot;small&quot; selected=&quot;selected&quot;&gt;Small Shirt&lt;/option&gt;<br />
  229. 229 &lt;option value=&quot;med&quot;&gt;Medium  Shirt&lt;/option&gt;<br />
  230. 230 &lt;option value=&quot;large&quot; selected=&quot;selected&quot;&gt;Large Shirt&lt;/option&gt;<br />
  231. 231 &lt;option value=&quot;xlarge&quot;&gt;Extra Large Shirt&lt;/option&gt;<br />
  232. 232 &lt;/select&gt;</code>
  233. 233 
  234. 234 
  235. 235 <p>If you would like the opening &lt;select> to contain additional data, like an <kbd>id</kbd> attribute or JavaScript, you can pass it as a string in the
  236. 236 fourth parameter:</p>
  237. 237 
  238. 238 <code>$js = 'id="shirts" onChange="some_function();"';<br />
  239. 239 <br />
  240. 240 echo form_dropdown('shirts', $options, 'large', $js);</code>
  241. 241 
  242. 242 <p>If the array passed as $options is a multidimensional array, form_dropdown() will produce an &lt;optgroup&gt; with the array key as the label.</p>
  243. 243 
  244. 244 <h2>form_multiselect()</h2>
  245. 245 
  246. 246 <p>Lets you create a standard multiselect field.  The first parameter will contain the name of the field,
  247. 247 the second parameter will contain an associative array of options, and the third parameter will contain the
  248. 248 value or values you wish to be selected.  The parameter usage is identical to using <kbd>form_dropdown()</kbd> above,
  249. 249 except of course that the name of the field will need to use POST array syntax, e.g. <samp>foo[]</samp>.</p>
  250. 250 
  251. 251 
  252. 252 <h2>form_fieldset()</h2>
  253. 253 
  254. 254 <p>Lets you generate fieldset/legend fields.</p>
  255. 255 <code>echo form_fieldset('Address Information');<br />
  256. 256 echo &quot;&lt;p&gt;fieldset content here&lt;/p&gt;\n&quot;;<br />
  257. 257 echo form_fieldset_close();
  258. 258 <br />
  259. 259 <br />
  260. 260 // Produces<br />
  261. 261 &lt;fieldset&gt;
  262. 262 <br />
  263. 263 &lt;legend&gt;Address Information&lt;/legend&gt;
  264. 264 <br />
  265. 265 &lt;p&gt;form content here&lt;/p&gt;
  266. 266 <br />
  267. 267 &lt;/fieldset&gt;</code>
  268. 268 <p>Similar to other functions, you can submit an associative array in the second parameter if you prefer to set additional attributes. </p>
  269. 269 <p><code>$attributes = array('id' =&gt; 'address_info', 'class' =&gt; 'address_info');<br />
  270. 270     echo form_fieldset('Address Information', $attributes);<br />
  271. 271 echo &quot;&lt;p&gt;fieldset content here&lt;/p&gt;\n&quot;;<br />
  272. 272 echo form_fieldset_close(); <br />
  273. 273 <br />
  274. 274 // Produces<br />
  275. 275 &lt;fieldset id=&quot;address_info&quot; class=&quot;address_info&quot;&gt; <br />
  276. 276 &lt;legend&gt;Address Information&lt;/legend&gt; <br />
  277. 277 &lt;p&gt;form content here&lt;/p&gt; <br />
  278. 278 &lt;/fieldset&gt;</code></p>
  279. 279 <h2>form_fieldset_close()</h2>
  280. 280 <p>Produces a closing &lt;/fieldset&gt; tag.  The only advantage to using this function is it permits you to pass data to it
  281. 281     which will be added below the tag.  For example:</p>
  282. 282 <code>$string = &quot;&lt;/div&gt;&lt;/div&gt;&quot;;<br />
  283. 283 <br />
  284. 284 echo form_fieldset_close($string);<br />
  285. 285 <br />
  286. 286 // Would produce:<br />
  287. 287 &lt;/fieldset&gt;<br />
  288. 288 &lt;/div&gt;&lt;/div&gt;</code>
  289. 289 <h2>form_checkbox()</h2>
  290. 290 <p>Lets you generate a checkbox field. Simple example:</p>
  291. 291 <code>echo form_checkbox('newsletter', 'accept', TRUE);<br />
  292. 292 <br />
  293. 293 // Would produce:<br />
  294. 294 <br />
  295. 295 &lt;input type=&quot;checkbox&quot; name=&quot;newsletter&quot; value=&quot;accept&quot; checked=&quot;checked&quot; /&gt;</code>
  296. 296 <p>The third parameter contains a boolean TRUE/FALSE to determine whether the box should be checked or not.</p>
  297. 297 <p>Similar to the other form functions in this helper, you can also pass an array of attributes to the function:</p>
  298. 298 
  299. 299 <code>$data = array(<br />
  300. 300 &nbsp;&nbsp;&nbsp;&nbsp;'name'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> 'newsletter',<br />
  301. 301 &nbsp;&nbsp;&nbsp;&nbsp;'id'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> 'newsletter',<br />
  302. 302 &nbsp;&nbsp;&nbsp;&nbsp;'value'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> 'accept',<br />
  303. 303 &nbsp;&nbsp;&nbsp;&nbsp;'checked'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> TRUE,<br />
  304. 304 &nbsp;&nbsp;&nbsp;&nbsp;'style'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; => 'margin:10px',<br />
  305. 305 &nbsp;&nbsp;&nbsp;&nbsp;);<br />
  306. 306 <br />
  307. 307 echo form_checkbox($data);<br />
  308. 308 <br />
  309. 309 // Would produce:<br /><br />
  310. 310 &lt;input type="checkbox" name="newsletter" id="newsletter" value="accept" checked="checked" style="margin:10px" /></code>
  311. 311 
  312. 312 <p>As with other functions, if you would like the tag to contain  additional data, like JavaScript, you can pass it as a string in the
  313. 313 fourth parameter:</p>
  314. 314 
  315. 315 <code>$js = 'onClick="some_function()"';<br />
  316. 316 <br />
  317. 317  echo form_checkbox('newsletter', 'accept', TRUE, $js)</code>
  318. 318 
  319. 319 
  320. 320 <h2>form_radio()</h2>
  321. 321 <p>This function is identical in all respects to the <dfn>form_checkbox()</dfn> function above except that is sets it as a "radio" type.</p>
  322. 322 
  323. 323 
  324. 324 <h2>form_submit()</h2>
  325. 325 
  326. 326 <p>Lets you generate a standard submit button. Simple example:</p>
  327. 327 <code>echo form_submit('mysubmit', 'Submit Post!');<br />
  328. 328 <br />
  329. 329 // Would produce:<br />
  330. 330 <br />
  331. 331 &lt;input type=&quot;submit&quot; name=&quot;mysubmit&quot; value=&quot;Submit Post!&quot; /&gt;</code>
  332. 332 <p>Similar to other functions, you can submit an associative array in the first parameter if you prefer to set your own attributes.
  333. 333     The third parameter lets you add extra data to your form, like JavaScript.</p>
  334. 334 <h2>form_label()</h2>
  335. 335 <p>Lets you generate a &lt;label&gt;. Simple example:</p>
  336. 336 <code>echo form_label('What is your Name', 'username');<br />
  337. 337 <br />
  338. 338 // Would produce:
  339. 339 <br />
  340. 340 &lt;label for=&quot;username&quot;&gt;What is your Name&lt;/label&gt;</code>
  341. 341 <p>Similar to other functions, you can submit an associative array in the third parameter if you prefer to set additional attributes.    </p>
  342. 342 <p><code>$attributes = array(<br />
  343. 343 &nbsp;&nbsp;&nbsp;&nbsp;'class' =&gt; 'mycustomclass',<br />
  344. 344 &nbsp;&nbsp;&nbsp;&nbsp;'style' =&gt; 'color: #000;',<br />
  345. 345 );<br />
  346. 346     echo form_label('What is your Name', 'username', $attributes);<br />
  347. 347         <br />
  348. 348 // Would produce: <br />
  349. 349 &lt;label for=&quot;username&quot; class=&quot;mycustomclass&quot; style=&quot;color: #000;&quot;&gt;What is your Name&lt;/label&gt;</code></p>
  350. 350 <h2>form_reset()</h2>
  351. 351 
  352. 352 <p>Lets you generate a standard reset button. Use is identical to <dfn>form_submit()</dfn>.</p>
  353. 353 
  354. 354 <h2>form_button()</h2>
  355. 355 
  356. 356 <p>Lets you generate a standard button element. You can minimally pass the button name and content in the first and second parameter:</p>
  357. 357 <code>
  358. 358 echo form_button('name','content');<br />
  359. 359 <br />
  360. 360 // Would produce<br />
  361. 361 &lt;button name="name" type="button"&gt;Content&lt;/button&gt;
  362. 362 </code>
  363. 363 
  364. 364 Or you can pass an associative array containing any data you wish your form to contain:
  365. 365 <code>
  366. 366 $data = array(<br />
  367. 367 &nbsp;&nbsp;&nbsp;&nbsp;'name' => 'button',<br />
  368. 368 &nbsp;&nbsp;&nbsp;&nbsp;'id' => 'button',<br />
  369. 369 &nbsp;&nbsp;&nbsp;&nbsp;'value' => 'true',<br />
  370. 370 &nbsp;&nbsp;&nbsp;&nbsp;'type' => 'reset',<br />
  371. 371 &nbsp;&nbsp;&nbsp;&nbsp;'content' => 'Reset'<br />
  372. 372 );<br />
  373. 373 <br />
  374. 374 echo form_button($data);<br />
  375. 375 <br />
  376. 376 // Would produce:<br />
  377. 377 &lt;button name="button" id="button" value="true" type="reset"&gt;Reset&lt;/button&gt;
  378. 378 </code>
  379. 379 
  380. 380 If you would like your form to contain some additional data, like JavaScript, you can pass it as a string in the third parameter:
  381. 381 <code>
  382. 382 $js = 'onClick="some_function()"';<br /><br />
  383. 383 echo form_button('mybutton', 'Click Me', $js);
  384. 384 </code>
  385. 385 
  386. 386 
  387. 387 <h2>form_close()</h2>
  388. 388 
  389. 389 <p>Produces a closing &lt;/form> tag.  The only advantage to using this function is it permits you to pass data to it
  390. 390 which will be added below the tag.  For example:</p>
  391. 391 
  392. 392 <code>$string = "&lt;/div>&lt;/div>";<br />
  393. 393 <br />
  394. 394 echo form_close($string);<br />
  395. 395 <br />
  396. 396 // Would produce:<br />
  397. 397 <br />
  398. 398 &lt;/form><br />
  399. 399 &lt;/div>&lt;/div></code>
  400. 400 
  401. 401 
  402. 402 
  403. 403 
  404. 404 
  405. 405 <h2>form_prep()</h2>
  406. 406 
  407. 407 <p>Allows you to safely use HTML and characters such as quotes within form elements without breaking out of the form.  Consider this example:</p>
  408. 408 
  409. 409 <code>$string = 'Here is a string containing <strong>"quoted"</strong> text.';<br />
  410. 410 <br />
  411. 411 &lt;input type="text" name="myform" value="<var>$string</var>" /></code>
  412. 412 
  413. 413 <p>Since the above string contains a set of quotes it will cause the form to break.
  414. 414 The form_prep function converts HTML so that it can be used safely:</p>
  415. 415 
  416. 416 <code>&lt;input type="text" name="myform" value="<var>&lt;?php echo form_prep($string); ?></var>" /></code>
  417. 417 
  418. 418 <p class="important"><strong>Note:</strong> If you use any of the form helper functions listed in this page the form
  419. 419 values will be prepped automatically, so there is no need to call this function. Use it only if you are
  420. 420 creating your own form elements.</p>

  422. 422 
  423. 423 <h2>set_value()</h2>
  424. 424 
  425. 425 <p>Permits you to set the value of an input form or textarea. You must supply the field name via the first parameter of the function.
  426. 426 The second (optional) parameter allows you to set a default value for the form. Example:</p>
  427. 427 
  428. 428 <code>&lt;input type="text" name="quantity" value="<dfn>&lt;?php echo set_value('quantity', '0'); ?></dfn>" size="50" /></code>
  429. 429 
  430. 430 <p>The above form will show "0" when loaded for the first time.</p>

  432. 432 <h2>set_select()</h2>
  433. 433 
  434. 434 <p>If you use a <dfn>&lt;select></dfn> menu, this function permits you to display the menu item that was selected.  The first parameter
  435. 435 must contain the name of the select menu, the second parameter must contain the value of
  436. 436 each item, and the third (optional) parameter lets you set an item as the default (use boolean TRUE/FALSE).</p>
  437. 437 
  438. 438 <p>Example:</p>
  439. 439 
  440. 440 <code>
  441. 441 &lt;select name="myselect"><br />
  442. 442 &lt;option value="one" <dfn>&lt;?php echo  set_select('myselect', 'one', TRUE); ?></dfn> >One&lt;/option><br />
  443. 443 &lt;option value="two" <dfn>&lt;?php echo  set_select('myselect', 'two'); ?></dfn> >Two&lt;/option><br />
  444. 444 &lt;option value="three" <dfn>&lt;?php echo  set_select('myselect', 'three'); ?></dfn> >Three&lt;/option><br />
  445. 445 &lt;/select>
  446. 446 </code>

  449. 449 <h2>set_checkbox()</h2>
  450. 450 
  451. 451 <p>Permits you to display a checkbox in the state it was submitted.  The first parameter
  452. 452 must contain the name of the checkbox, the second parameter must contain its value, and the third (optional) parameter lets you set an item as the default (use boolean TRUE/FALSE). Example:</p>
  453. 453 
  454. 454 <code>&lt;input type="checkbox" name="mycheck" value="1" <dfn>&lt;?php echo set_checkbox('mycheck', '1'); ?></dfn> /><br />
  455. 455 &lt;input type="checkbox" name="mycheck" value="2" <dfn>&lt;?php echo set_checkbox('mycheck', '2'); ?></dfn> /></code>

  460. 460 <p>Permits you to display radio buttons in the state they were submitted. This function is identical to the <strong>set_checkbox()</strong> function above.</p>
  461. 461 
  462. 462 <code>&lt;input type="radio" name="myradio" value="1" <dfn>&lt;?php echo  set_radio('myradio', '1', TRUE); ?></dfn> /><br />
  463. 463 &lt;input type="radio" name="myradio" value="2" <dfn>&lt;?php echo  set_radio('myradio', '2'); ?></dfn> /></code>
