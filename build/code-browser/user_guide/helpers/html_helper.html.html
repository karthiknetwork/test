  59.  59 <h1>HTML Helper</h1>
  60.  60 
  61.  61 <p>The HTML Helper file contains functions that assist in working with HTML.</p>
  62.  62 
  63.  63 <ul>
  64.  64  <li><a href="#br">br()</a></li>
  65.  65  <li><a href="#heading">heading()</a></li>
  66.  66  <li><a href="#img">img()</a></li>
  67.  67  <li><a href="#link_tag">link_tag()</a></li>
  68.  68  <li><a href="#nbs">nbs()</a></li>
  69.  69  <li><a href="#ol_and_ul">ol() and ul()</a></li>
  70.  70  <li><a href="#meta">meta()</a></li>
  71.  71  <li><a href="#doctype">doctype()</a></li>
  72.  72 </ul>
  73.  73 
  74.  74 <h2>Loading this Helper</h2>
  75.  75 
  76.  76 <p>This helper is loaded using the following code:</p>
  77.  77 <code>$this->load->helper('html');</code>
  78.  78 
  79.  79 <p>The following functions are available:</p>
  80.  80 
  81.  81 <h2><a name="br"></a>br()</h2>
  82.  82 <p>Generates line break tags (&lt;br />) based on the number you submit. Example:</p>
  83.  83 <code>echo br(3);</code>
  84.  84 <p>The above would produce: &lt;br />&lt;br />&lt;br /></p>
  85.  85 
  86.  86 <h2><a name="heading"></a>heading()</h2>
  87.  87 <p>Lets you create HTML &lt;h1> tags.  The first parameter will contain the data, the
  88.  88 second the size of the heading.  Example:</p>
  89.  89 <code>echo heading('Welcome!', 3);</code>
  90.  90 <p>The above would produce:  &lt;h3>Welcome!&lt;/h3></p>
  91.  91 
  92.  92 <p>Additionally, in order to add attributes to the heading tag such as HTML classes, ids or inline styles, a third parameter is available.</p>
  93.  93 <code>echo heading('Welcome!', 3, 'class="pink"')</code>
  94.  94 <p>The above code produces: &lt;h3 class="pink">Welcome!<&lt;h3></p>
  95.  95 
  96.  96 
  97.  97 <h2><a name="img"></a>img()</h2>
  98.  98 <p>Lets you create HTML &lt;img /&gt; tags.  The first parameter  contains the image source. Example:</p>
  99.  99 <code>echo img('images/picture.jpg');<br />
  100. 100 // gives &lt;img src=&quot;http://site.com/images/picture.jpg&quot; /&gt;</code>
  101. 101 <p>There is an optional second parameter that is a TRUE/FALSE value that specifics if the src should have the page specified by $config['index_page'] added to the address it creates. Presumably, this would be if you were using a media controller.</p>
  102. 102 <p><code>echo img('images/picture.jpg', TRUE);<br />
  103. 103 // gives &lt;img src=&quot;http://site.com/index.php/images/picture.jpg&quot; alt=&quot;&quot; /&gt;</code></p>
  104. 104 <p>Additionally, an associative array can be passed to the img() function for complete control over all attributes and values. If an alt attribute is not provided, CodeIgniter will generate an empty string.</p>
  105. 105 <p><code> $image_properties = array(<br />
  106. 106  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'src' =&gt; 'images/picture.jpg',<br />
  107. 107  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'alt' =&gt; 'Me, demonstrating how to eat 4 slices of pizza at one time',<br />
  108. 108  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'class' =&gt; 'post_images',<br />
  109. 109  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'width' =&gt; '200',<br />
  110. 110  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'height' =&gt; '200',<br />
  111. 111  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'title' =&gt; 'That was quite a night',<br />
  112. 112  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'rel' =&gt; 'lightbox',<br />
  113. 113  );<br />
  114. 114  <br />
  115. 115  img($image_properties);<br />
  116. 116  // &lt;img src=&quot;http://site.com/index.php/images/picture.jpg&quot; alt=&quot;Me, demonstrating how to eat 4 slices of pizza at one time&quot; class=&quot;post_images&quot; width=&quot;200&quot; height=&quot;200&quot; title=&quot;That was quite a night&quot; rel=&quot;lightbox&quot; /&gt;</code></p>
  117. 117 
  118. 118 <h2><a name="link_tag"></a>link_tag()</h2>
  119. 119 <p>Lets you create HTML &lt;link /> tags.  This is useful for stylesheet links, as well as other links. The parameters are href, with optional rel, type, title, media and index_page. index_page is a TRUE/FALSE value that specifics if the href should have the page specified by $config['index_page'] added to the address it creates.<code>
  120. 120 echo link_tag('css/mystyles.css');<br />
  121. 121 // gives &lt;link href=&quot;http://site.com/css/mystyles.css&quot; rel=&quot;stylesheet&quot; type=&quot;text/css&quot; /&gt;</code></p>
  122. 122 <p>Further examples:</p>
  123. 123 
  124. 124 <code>
  125. 125  echo link_tag('favicon.ico', 'shortcut icon', 'image/ico');<br />
  126. 126  // &lt;link href=&quot;http://site.com/favicon.ico&quot; rel=&quot;shortcut icon&quot; type=&quot;image/ico&quot; /&gt;
  127. 127  <br />
  128. 128  <br />
  129. 129  echo link_tag('feed', 'alternate', 'application/rss+xml', 'My RSS Feed');<br />
  130. 130  // &lt;link href=&quot;http://site.com/feed&quot; rel=&quot;alternate&quot; type=&quot;application/rss+xml&quot; title=&quot;My RSS Feed&quot; /&gt; </code>
  131. 131 <p>Additionally, an associative array can be passed to the link() function for complete control over all attributes and values.</p>
  132. 132 <p><code>
  133. 133  $link = array(<br />
  134. 134  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'href' =&gt; 'css/printer.css',<br />
  135. 135  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'rel' =&gt; 'stylesheet',<br />
  136. 136  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'type' =&gt; 'text/css',<br />
  137. 137  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'media' =&gt; 'print'<br />
  138. 138  );<br />
  139. 139  <br />
  140. 140  echo link_tag($link);<br />
  141. 141  // &lt;link href=&quot;http://site.com/css/printer.css&quot; rel=&quot;stylesheet&quot; type=&quot;text/css&quot; media=&quot;print&quot; /&gt;</code></p>
  142. 142 
  143. 143 <h2><a name="nbs"></a>nbs()</h2>
  144. 144 <p>Generates non-breaking spaces (&amp;nbsp;) based on the number you submit. Example:</p>
  145. 145 <code>echo nbs(3);</code>
  146. 146 <p>The above would produce: &amp;nbsp;&amp;nbsp;&amp;nbsp;</p>
  147. 147 
  148. 148 <h2><a name="ol_and_ul"></a>ol()&nbsp; and&nbsp; ul()</h2>
  149. 149 
  150. 150 <p>Permits you to generate ordered or unordered HTML lists from simple or multi-dimensional arrays. Example:</p>
  151. 151 
  152. 152 <code>
  153. 153 $this->load->helper('html');<br />
  154. 154 <br />
  155. 155 $list = array(<br />
  156. 156 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'red', <br />
  157. 157 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'blue', <br />
  158. 158 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'green',<br />
  159. 159 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'yellow'<br />
  160. 160 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;);<br />
  161. 161 <br />
  162. 162 $attributes = array(<br />
  163. 163 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'class' => 'boldlist',<br />
  164. 164 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'id'&nbsp;&nbsp;&nbsp;&nbsp;=> 'mylist'<br />
  165. 165 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;);<br />
  166. 166 <br />
  167. 167 echo ul($list, $attributes);<br />
  168. 168 </code>
  169. 169 
  170. 170 <p>The above code will produce this:</p>
  171. 171 
  172. 172 <code>
  173. 173 &lt;ul&nbsp;class="boldlist"&nbsp;id="mylist"><br />
  174. 174 &nbsp;&nbsp;&lt;li>red&lt;/li><br />
  175. 175 &nbsp;&nbsp;&lt;li>blue&lt;/li><br />
  176. 176 &nbsp;&nbsp;&lt;li>green&lt;/li><br />
  177. 177 &nbsp;&nbsp;&lt;li>yellow&lt;/li><br />
  178. 178 &lt;/ul>

  272. 272 <h2><a name="meta"></a>meta()</h2>
  273. 273 
  274. 274 <p>Helps you generate meta tags.  You can pass strings to the function, or simple arrays, or multidimensional ones. Examples:</p>
  275. 275 
  276. 276 <code>
  277. 277 echo meta('description', 'My Great site');<br />
  278. 278 // Generates:  &lt;meta name="description" content="My Great Site" /><br />
  279. 279 <br /><br />
  280. 280 
  281. 281 echo meta('Content-type', 'text/html; charset=utf-8', 'equiv'); // Note the third parameter.  Can be "equiv" or "name"<br />
  282. 282 // Generates:  &lt;meta http-equiv="Content-type" content="text/html; charset=utf-8" /><br />
  283. 283 
  284. 284 <br /><br />
  285. 285 
  286. 286 echo meta(array('name' => 'robots', 'content' => 'no-cache'));<br />
  287. 287 // Generates:  &lt;meta name="robots" content="no-cache" /><br />
  288. 288 
  289. 289 <br /><br />
  290. 290 
  291. 291 $meta = array(<br />
  292. 292 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;array('name' => 'robots', 'content' => 'no-cache'),<br />
  293. 293 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;array('name' => 'description', 'content' => 'My Great Site'),<br />
  294. 294 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;array('name' => 'keywords', 'content' => 'love, passion, intrigue, deception'),<br />
  295. 295 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;array('name' => 'robots', 'content' => 'no-cache'),<br />
  296. 296 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;array('name' => 'Content-type', 'content' => 'text/html; charset=utf-8', 'type' => 'equiv')<br />
  297. 297 &nbsp;&nbsp;&nbsp;&nbsp;);<br />
  298. 298 <br />
  299. 299 echo meta($meta);
  300. 300 <br />
  301. 301 // Generates:  <br />
  302. 302 // &lt;meta name="robots" content="no-cache" /><br />
  303. 303 // &lt;meta name="description" content="My Great Site" /><br />
  304. 304 // &lt;meta name="keywords" content="love, passion, intrigue, deception" /><br />
  305. 305 // &lt;meta name="robots" content="no-cache" /><br />
  306. 306 // &lt;meta http-equiv="Content-type" content="text/html; charset=utf-8" />
  307. 307 </code>

  310. 310 <h2><a name="doctype"></a>doctype()</h2>
