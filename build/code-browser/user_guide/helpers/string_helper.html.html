  59.  59 <h1>String Helper</h1>
  60.  60 
  61.  61 <p>The String Helper file contains functions that assist in working with strings.</p>
  62.  62 
  63.  63 
  64.  64 <h2>Loading this Helper</h2>
  65.  65 
  66.  66 <p>This helper is loaded using the following code:</p>
  67.  67 <code>$this->load->helper('string');</code>
  68.  68 
  69.  69 <p>The following functions are available:</p>
  70.  70 
  71.  71 <h2>random_string()</h2>
  72.  72 
  73.  73 <p>Generates a random string based on the type and length you specify.  Useful for creating passwords or generating random hashes.</p>
  74.  74 
  75.  75 <p>The first parameter specifies the type of string, the second parameter specifies the length.  The following choices are available:</p>
  76.  76 
  77.  77  alpha, alunum, numeric, nozero, unique, md5, encrypt and sha1
  78.  78 <ul>
  79.  79  <li><strong>alpha</strong>:&nbsp; A string with lower and uppercase letters only.</li>
  80.  80  <li><strong>alnum</strong>:&nbsp; Alpha-numeric string with lower and uppercase characters.</li>
  81.  81  <li><strong>numeric</strong>:&nbsp; Numeric string.</li>
  82.  82  <li><strong>nozero</strong>:&nbsp; Numeric string with no zeros.</li>
  83.  83  <li><strong>unique</strong>:&nbsp; Encrypted with MD5 and uniqid(). Note: The length parameter is not available for this type.
  84.  84  Returns a fixed length 32 character string.</li>
  85.  85  <li><strong>sha1</strong>:&nbsp; An encrypted random number based on <kbd>do_hash()</kbd> from the <a href="security_helper.html">security helper</a>.</li>
  86.  86 </ul>
  87.  87 
  88.  88 <p>Usage example:</p>
  89.  89 
  90.  90 <code>echo random_string('alnum', 16);</code>
  91.  91 
  92.  92 
  93.  93 <h2>increment_string()</h2>
  94.  94 
  95.  95 <p>Increments a string by appending a number to it or increasing the number. Useful for creating "copies" or a file or duplicating database content which has unique titles or slugs.</p>
  96.  96 
  97.  97 <p>Usage example:</p>
  98.  98 
  99.  99 <code>echo increment_string('file', '_'); // "file_1"<br/>
  100. 100 echo increment_string('file', '-', 2); // "file-2"<br/>
  101. 101 echo increment_string('file-4'); // "file-5"<br/></code>
  102. 102 
  103. 103 
  104. 104 <h2>alternator()</h2>
  105. 105 
  106. 106 <p>Allows two or more items to be alternated between, when cycling through a loop.  Example:</p>
  107. 107 
  108. 108 <code>for ($i = 0; $i < 10; $i++)<br />
  109. 109 {<br />
  110. 110 &nbsp;&nbsp;&nbsp;&nbsp;echo alternator('string one', 'string two');<br />
  111. 111 }<br />
  112. 112 </code>
  113. 113 
  114. 114 <p>You can add as many parameters as you want, and with each iteration of your loop the next item will be returned.</p>
  115. 115 
  116. 116 <code>for ($i = 0; $i < 10; $i++)<br />
  117. 117 {<br />
  118. 118 &nbsp;&nbsp;&nbsp;&nbsp;echo alternator('one', 'two', 'three', 'four', 'five');<br />
  119. 119 }<br />
  120. 120 </code>
  121. 121 
  122. 122 <p><strong>Note:</strong> To use multiple separate calls to this function simply call the function with no arguments to re-initialize.</p>
  123. 123 
  124. 124 
  125. 125 
  126. 126 <h2>repeater()</h2>
  127. 127 <p>Generates repeating copies of the data you submit. Example:</p>
  128. 128 <code>$string = "\n";<br />
  129. 129 echo repeater($string, 30);</code>
  130. 130 
  131. 131 <p>The above would generate 30 newlines.</p>
  132. 132 <h2>reduce_double_slashes()</h2>
  133. 133 <p>Converts double slashes in a string to a single slash, except those found in http://. Example: </p>
  134. 134 <code>$string = &quot;http://example.com//index.php&quot;;<br />
  135. 135 echo reduce_double_slashes($string); // results in &quot;http://example.com/index.php&quot;</code>
  136. 136 <h2>trim_slashes()</h2>
  137. 137 <p>Removes any leading/trailing slashes from a string. Example:<br />
  138. 138  <br />
  139. 139  <code>$string = &quot;/this/that/theother/&quot;;<br />
  140. 140 echo trim_slashes($string); // results in this/that/theother</code></p>
  141. 141 
  142. 142 
  143. 143 <h2>reduce_multiples()</h2>
  144. 144 <p>Reduces multiple instances of a particular character occuring directly after each other. Example:</p>
  145. 145 <code>
  146. 146 $string="Fred, Bill,, Joe, Jimmy";<br />
  147. 147 $string=reduce_multiples($string,","); //results in "Fred, Bill, Joe, Jimmy"
  148. 148 </code>
  149. 149 <p>The function accepts the following parameters:
  150. 150 <code>reduce_multiples(string: text to search in, string: character to reduce, boolean: whether to remove the character from the front and end of the string)</code>
  151. 151 
  152. 152 The first parameter contains the string in which you want to reduce the multiplies. The second parameter contains the character you want to have reduced.
  153. 153 The third parameter is FALSE by default; if set to TRUE it will remove occurences of the character at the beginning and the end of the string. Example:
  154. 154 
  155. 155 <code>
  156. 156 $string=",Fred, Bill,, Joe, Jimmy,";<br />
  157. 157 $string=reduce_multiples($string, ", ", TRUE); //results in "Fred, Bill, Joe, Jimmy"
  158. 158 </code>
  159. 159 </p>
  160. 160 
  161. 161 <h2>quotes_to_entities()</h2>
  162. 162 <p>Converts single and double quotes in a string to the corresponding HTML entities. Example:</p>
  163. 163 <code>$string="Joe's \"dinner\"";<br />
  164. 164 $string=quotes_to_entities($string); //results in "Joe&amp;#39;s &amp;quot;dinner&amp;quot;"
  165. 165 </code>
  166. 166 
  167. 167 <h2>strip_quotes()</h2>
  168. 168 <p>Removes single and double quotes from a string. Example:</p>
  169. 169 <code>$string="Joe's \"dinner\"";<br />
  170. 170 $string=strip_quotes($string); //results in "Joes dinner"
  171. 171 </code>
