  58.  58 <h1>Installation Instructions</h1>
  59.  59 
  60.  60 <p>CodeIgniter is installed in four steps:</p>
  61.  61 
  62.  62 <ol>
  63.  63 <li>Unzip the package.</li>
  64.  64 <li>Upload the CodeIgniter folders and files to your server.  Normally the index.php file will be at your root.</li>
  65.  65 <li>Open the <dfn>application/config/config.php</dfn> file with a text editor and set your base URL.  If you intend to use encryption or sessions, set your encryption key.</li>
  66.  66 <li>If you intend to use a database, open the <dfn>application/config/database.php</dfn> file with a text editor and set your database settings.</li>
  67.  67 </ol>
  68.  68 
  69.  69 <p>If you wish to increase security by hiding the location of your CodeIgniter files you can rename the <dfn>system</dfn> and <dfn>application</dfn> folders
  70.  70 to something more private.  If you do rename them, you must open your main <kbd>index.php</kbd> file and set the <samp>$system_folder</samp> and <samp>$application_folder</samp>
  71.  71 variables at the top of the file with the new name you've chosen.</p>
  72.  72 
  73.  73 <p>For the best security, both the <dfn>system</dfn> and any <dfn>application</dfn> folders should be placed above web root so that they are not directly accessible via a browser.  By default, .htaccess files are included in each folder to help prevent direct access, but it is best to remove them from public access entirely in case the web server configuration changes or doesn't abide by the .htaccess.</p>
  74.  74 
  75.  75 <p>After moving them, open your main <kdb>index.php</kbd> file and set the <samp>$system_folder</samp> and <samp>$application_folder</samp> variables, preferably with a full path, e.g. '<dfn>/www/MyUser/system</dfn>'.</p>
  76.  76 
  77.  77 <p>
  78.  78     One additional measure to take in production environments is to disable
  79.  79     PHP error reporting and any other development-only functionality. In CodeIgniter,
  80.  80     this can be done by setting the <kbd>ENVIRONMENT</kbd> constant, which is
  81.  81     more fully described on the <a href="../general/security.html">security page</a>.
  82.  82 </p>
  83.  83 
  84.  84 <p>That's it!</p>
  85.  85 
  86.  86 <p>If you're new to CodeIgniter, please read the <a href="../overview/getting_started.html">Getting Started</a> section of the User Guide to begin learning how
  87.  87 to build dynamic PHP applications. Enjoy!</p>
