  59.  59 <h1>Caching Driver</h1>
  60.  60 
  61.  61 <p>CodeIgniter features wrappers around some of the most popular forms of fast and dynamic caching.  All but file-based caching require specific server requirements, and a Fatal Exception will be thrown if server requirements are not met.</p>
  62.  62

  67.  67 </ul>

  69.  69 <h3>Available Drivers</h3>
  70.  70 <ul>
  71.  71  <li><a href="#apc" title="APC Cache">Alternative PHP Cache (APC) Caching</a></li>

  77.  77 <h2 id="example_usage">Example Usage</h2>
  78.  78 
  79.  79 <p>The following example will load the cache driver, specify <a href="#apc" title="APC">APC</a> as the driver to use, and fall back to file-based caching if APC is not available in the hosting environment.</p>
  80.  80 
  81.  81 <code>
  82.  82 $this->load->driver('cache', array('adapter' => 'apc', 'backup' => 'file'));<br />
  83.  83 <br />
  84.  84 if ( ! $foo = $this->cache->get('foo'))<br />
  85.  85 {<br />
  86.  86 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;echo 'Saving to the cache!&lt;br />';<br />
  87.  87 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;$foo = 'foobarbaz!';<br />
  88.  88 <br />
  89.  89 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;// Save into the cache for 5 minutes<br />
  90.  90 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;$this->cache->save('foo', $foo, 300);<br />
  91.  91 }<br />
  92.  92 <br />
  93.  93 echo $foo;
  94.  94 </code>

  96.  96 <h1 id="function_reference">Function Reference</h1>
  97.  97 
  98.  98 <h2>is_supported(<var>driver</var>['string'])</h2>
  99.  99 
  100. 100 <p>This function is automatically called when accessing drivers via <samp>$this->cache->get()</samp>.  However, if the individual drivers are used, make sure to call this function to ensure the driver is supported in the hosting environment.</p>
  101. 101 
  102. 102 <code>
  103. 103 if ($this->cache->apc->is_supported())<br />
  104. 104 {<br />
  105. 105 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;if ($data = $this->cache->apc->get('my_cache'))<br />
  106. 106 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;{<br />
  107. 107 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;// do things.<br />
  108. 108 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;}<br />
  109. 109 }
  110. 110 </code>
  111. 111 
  112. 112 <h2>get(<var>id</var>['string'])</h2>
  113. 113 
  114. 114 <p>This function will attempt to fetch an item from the cache store.  If the item does not exist, the function will return <samp>FALSE</samp>.</p>
  115. 115 <code>$foo = $this->cache->get('my_cached_item');</code>
  116. 116 
  117. 117 <h2>save(<var>id</var>['string'], <var>data</var>['mixed'], <var>ttl</var>['int'])</h2>
  118. 118 
  119. 119 <p>This function will save an item to the cache store.  If saving fails, the function will return <samp>FALSE</samp>.</p>
  120. 120 <p>The optional third parameter (Time To Live) defaults to 60 seconds.</p>
  121. 121 <code>$this->cache->save('cache_item_id', 'data_to_cache');</code>
  122. 122 
  123. 123 <h2>delete(<var>id</var>['string'])</h2>
  124. 124 
  125. 125 <p>This function will delete a specific item from the cache store.  If item deletion fails, the function will return <samp>FALSE</samp>.</p>
  126. 126 <code>$this->cache->delete('cache_item_id');</code>
  127. 127 
  128. 128 <h2>clean()</h2>
  129. 129 
  130. 130 <p>This function will 'clean' the entire cache.  If the deletion of the cache files fails, the function will return <samp>FALSE</samp>.</p>
  131. 131 
  132. 132 <code>$this->cache->clean();</code>
  133. 133 
  134. 134 <h2>cache_info()</h2>
  135. 135 
  136. 136 <p>This function will return information on the entire cache.</p>
  137. 137 
  138. 138 <code>var_dump($this->cache->cache_info());</code>
  139. 139 
  140. 140 <h2>get_metadata(<var>id</var>['string'])</h2>
  141. 141 
  142. 142 <p>This function will return detailed information on a specific item in the cache.</p>
  143. 143 
  144. 144 <code>var_dump($this->cache->get_metadata('my_cached_item'));</code>
  145. 145 
  146. 146 <h1>Drivers</h1>
  147. 147 
  148. 148 <h2 id="apc">Alternative PHP Cache (APC) Caching</h2>
  149. 149 
  150. 150 <p>All of the functions listed above can be accessed without passing a specific adapter to the driver loader as follows:</p>
  151. 151 <code>$this->load->driver('cache');<br />
  152. 152  $this->cache->apc->save('foo', 'bar', 10);</code>
  153. 153 <p>For more information on APC, please see <a href="http://php.net/apc">http://php.net/apc</a></p>
  154. 154 
  155. 155 <h2 id="file">File-based Caching</h2>
  156. 156 
  157. 157 <p>Unlike caching from the Output Class, the driver file-based caching allows for pieces of view files to be cached.  Use this with care, and make sure to benchmark your application, as a point can come where disk I/O will negate positive gains by caching.</p>
  158. 158 
  159. 159 <p>All of the functions listed above can be accessed without passing a specific adapter to the driver loader as follows:</p>
  160. 160 <code>$this->load->driver('cache');<br />
  161. 161  $this->cache->file->save('foo', 'bar', 10);</code>
  162. 162 
  163. 163 <h2 id="memcached">Memcached Caching</h2>
  164. 164 
  165. 165 <p>Multiple Memcached servers can be specified in the memcached.php configuration file, located in the <samp>application/config/</samp> directory.
  166. 166 
  167. 167 <p>All of the functions listed above can be accessed without passing a specific adapter to the driver loader as follows:</p>
  168. 168 <code>$this->load->driver('cache');<br />
  169. 169  $this->cache->memcached->save('foo', 'bar', 10);</code>
  170. 170 
  171. 171 <p>For more information on Memcached, please see <a href="http://php.net/memcached">http://php.net/memcached</a></p>
  172. 172 
  173. 173 <h2 id="dummy">Dummy Cache</h2>
  174. 174 
  175. 175 <p>This is a caching backend that will always 'miss.'  It stores no data, but lets you keep your caching code in place in environments that don't support your chosen cache.</p>
