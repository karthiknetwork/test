  58.  58 <h1>HTML Table Class</h1>
  59.  59 
  60.  60 <p>The Table Class provides functions that enable you to auto-generate HTML tables from arrays or database result sets.</p>
  61.  61 
  62.  62 <h2>Initializing the Class</h2>
  63.  63 
  64.  64 <p>Like most other classes in CodeIgniter, the Table class is initialized in your controller using the <dfn>$this->load->library</dfn> function:</p>
  65.  65 
  66.  66 <code>$this->load->library('table');</code>
  67.  67 <p>Once loaded, the Table library object will be available using: <dfn>$this->table</dfn></p>
  68.  68 
  69.  69 
  70.  70 <h2>Examples</h2>
  71.  71 
  72.  72 <p>Here is an example showing how you can create a table from a multi-dimensional array.
  73.  73 Note that the first array index will become the table heading (or you can set your own headings using the
  74.  74 <dfn>set_heading()</dfn> function described in the function reference below).</p>
  75.  75 
  76.  76 <code>
  77.  77 $this->load->library('table');<br />
  78.  78 <br />
  79.  79 $data = array(<br />
  80.  80 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;array('Name', 'Color', 'Size'),<br />
  81.  81 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;array('Fred', 'Blue', 'Small'),<br />
  82.  82 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;array('Mary', 'Red', 'Large'),<br />
  83.  83 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;array('John', 'Green', 'Medium') <br />
  84.  84 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;);<br />
  85.  85 <br />
  86.  86 echo $this->table->generate($data);
  87.  87 </code>
  88.  88 
  89.  89 <p>Here is an example of a table created from a database query result. The table class will automatically generate the
  90.  90 headings based on the table names (or you can set your own headings using the <dfn>set_heading()</dfn> function described
  91.  91 in the function reference below).</p>
  92.  92 
  93.  93 <code>
  94.  94 $this->load->library('table');<br />
  95.  95 <br />
  96.  96 $query = $this->db->query("SELECT * FROM my_table");<br />
  97.  97 <br />
  98.  98 echo $this->table->generate($query);
  99.  99 </code>
  100. 100 
  101. 101 
  102. 102 <p>Here is an example showing how you might create a table using discrete parameters:</p>
  103. 103 
  104. 104 <code>
  105. 105 $this->load->library('table');<br />
  106. 106 <br />
  107. 107 $this->table->set_heading('Name', 'Color', 'Size');<br />
  108. 108 <br />
  109. 109 $this->table->add_row('Fred', 'Blue', 'Small');<br />
  110. 110 $this->table->add_row('Mary', 'Red', 'Large');<br />
  111. 111 $this->table->add_row('John', 'Green', 'Medium');<br />
  112. 112 <br />
  113. 113 echo $this->table->generate();
  114. 114 </code>
  115. 115 
  116. 116 <p>Here is the same example, except instead of individual parameters, arrays are used:</p>
  117. 117 
  118. 118 <code>
  119. 119 $this->load->library('table');<br />
  120. 120 <br />
  121. 121 $this->table->set_heading(array('Name', 'Color', 'Size'));<br />
  122. 122 <br />
  123. 123 $this->table->add_row(array('Fred', 'Blue', 'Small'));<br />
  124. 124 $this->table->add_row(array('Mary', 'Red', 'Large'));<br />
  125. 125 $this->table->add_row(array('John', 'Green', 'Medium'));<br />
  126. 126 <br />
  127. 127 echo $this->table->generate();
  128. 128 </code>
  129. 129 
  130. 130 
  131. 131 <h2>Changing the Look of Your Table</h2>
  132. 132 
  133. 133 <p>The Table Class permits you to set a table template with which you can specify the design of your layout.  Here is the template
  134. 134 prototype:</p>
  135. 135 
  136. 136 <code>
  137. 137 $tmpl =  array (<br />
  138. 138 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'table_open'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;table border="0" cellpadding="4" cellspacing="0">',<br />
  139. 139 <br />
  140. 140 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'heading_row_start'&nbsp;&nbsp;&nbsp;=> '&lt;tr>',<br />
  141. 141 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'heading_row_end'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;/tr>',<br />
  142. 142 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'heading_cell_start'&nbsp;&nbsp;=> '&lt;th>',<br />
  143. 143 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'heading_cell_end'&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;/th>',<br />
  144. 144 <br />
  145. 145 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'row_start'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;tr>',<br />
  146. 146 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'row_end'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;/tr>',<br />
  147. 147 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'cell_start'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;td>',<br />
  148. 148 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'cell_end'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;/td>',<br />
  149. 149 <br />
  150. 150 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'row_alt_start'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;tr>',<br />
  151. 151 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'row_alt_end'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;/tr>',<br />
  152. 152 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'cell_alt_start'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;td>',<br />
  153. 153 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'cell_alt_end'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;/td>',<br />
  154. 154 <br />
  155. 155 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'table_close'&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;=> '&lt;/table>'<br />
  156. 156 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;);<br />
  157. 157 
  158. 158 <br />
  159. 159 $this->table->set_template($tmpl);
  160. 160 </code>
  161. 161 
  162. 162 <p class="important"><strong>Note:</strong>&nbsp; You'll notice there are two sets of "row" blocks in the template.  These permit you to create alternating row colors or design elements that alternate with each
  163. 163 iteration of the row data.</p>
  164. 164 
  165. 165 <p>You are NOT required to submit a complete template.  If you only need to change parts of the layout you can simply submit those elements.
  166. 166 In this example, only the table opening tag is being changed:</p>
  167. 167 
  168. 168 <code>
  169. 169 $tmpl =  array ( 'table_open'&nbsp;&nbsp;=> '&lt;table border="1" cellpadding="2" cellspacing="1" class="mytable">' );<br />
  170. 170 
  171. 171 <br />
  172. 172 $this->table->set_template($tmpl);
  173. 173 </code>
  174. 174 
  175. 175 <br />
  176. 176 <h1>Function Reference</h1>
  177. 177 
  178. 178 <h2>$this->table->generate()</h2>
  179. 179 <p>Returns a string containing the generated table.  Accepts an optional parameter which can be an array or a database result object.</p>
  180. 180 
  181. 181 <h2>$this->table->set_caption()</h2>
  182. 182 
  183. 183 <p>Permits you to add a caption to the table.</p>
  184. 184 
  185. 185 <code>$this->table->set_caption('Colors');</code>
  186. 186 
  187. 187 <h2>$this->table->set_heading()</h2>
  188. 188 
  189. 189 <p>Permits you to set the table heading.  You can submit an array or discrete params:</p>
  190. 190 
  191. 191 <code>$this->table->set_heading('Name', 'Color', 'Size');</code>
  192. 192 <code>$this->table->set_heading(array('Name', 'Color', 'Size'));</code>
  193. 193 
  194. 194 <h2>$this->table->add_row()</h2>
  195. 195 
  196. 196 <p>Permits you to add a row to your table.  You can submit an array or discrete params:</p>
  197. 197 
  198. 198 <code>$this->table->add_row('Blue', 'Red', 'Green');</code>
  199. 199 <code>$this->table->add_row(array('Blue', 'Red', 'Green'));</code>
  200. 200 
  201. 201 <p>If you would like to set an individual cell's tag attributes, you can use an associative array for that cell.  The associative key <dfn>'data'</dfn> defines the cell's data.  Any other key =&gt; val pairs are added as <dfn>key='val'</dfn> attributes to the tag:</p>
  202. 202 
  203. 203 <code>$cell = array('data' => 'Blue', 'class' => 'highlight', 'colspan' => 2);<br />
  204. 204 $this->table->add_row($cell, 'Red', 'Green');<br />
  205. 205 <br />
  206. 206 // generates<br />
  207. 207 // &lt;td class='highlight' colspan='2'&gt;Blue&lt;/td&gt;&lt;td&gt;Red&lt;/td&gt;&lt;td&gt;Green&lt;/td&gt;
  208. 208 </code>
  209. 209 
  210. 210 <h2>$this->table->make_columns()</h2>
  211. 211 
  212. 212 <p>This function takes a one-dimensional array as input and creates
  213. 213 a multi-dimensional array with a depth equal to the number of
  214. 214 columns desired.  This allows a single array with many elements to  be
  215. 215 displayed in a table that has a fixed column count.  Consider this example:</p>
  216. 216 
  217. 217 <code>
  218. 218 $list = array('one', 'two', 'three', 'four', 'five', 'six', 'seven', 'eight', 'nine', 'ten', 'eleven', 'twelve');<br />
  219. 219 <br />
  220. 220 $new_list = $this->table->make_columns($list, 3);<br />
  221. 221 <br />
  222. 222 $this->table->generate($new_list);<br />
  223. 223 <br />
  224. 224 // Generates a table with this prototype<br />
  225. 225 <br />
  226. 226 &lt;table border="0" cellpadding="4" cellspacing="0"&gt;<br />
  227. 227 &lt;tr&gt;<br />
  228. 228 &lt;td&gt;one&lt;/td&gt;&lt;td&gt;two&lt;/td&gt;&lt;td&gt;three&lt;/td&gt;<br />
  229. 229 &lt;/tr&gt;&lt;tr&gt;<br />
  230. 230 &lt;td&gt;four&lt;/td&gt;&lt;td&gt;five&lt;/td&gt;&lt;td&gt;six&lt;/td&gt;<br />
  231. 231 &lt;/tr&gt;&lt;tr&gt;<br />
  232. 232 &lt;td&gt;seven&lt;/td&gt;&lt;td&gt;eight&lt;/td&gt;&lt;td&gt;nine&lt;/td&gt;<br />
  233. 233 &lt;/tr&gt;&lt;tr&gt;<br />
  234. 234 &lt;td&gt;ten&lt;/td&gt;&lt;td&gt;eleven&lt;/td&gt;&lt;td&gt;twelve&lt;/td&gt;&lt;/tr&gt;<br />
  235. 235 &lt;/table&gt;</code>
  236. 236 
  237. 237 
  238. 238 
  239. 239 <h2>$this->table->set_template()</h2>
  240. 240 
  241. 241 <p>Permits you to set your template. You can submit a full or partial template.</p>
  242. 242 
  243. 243 <code>
  244. 244 $tmpl =  array ( 'table_open'&nbsp;&nbsp;=> '&lt;table border="1" cellpadding="2" cellspacing="1" class="mytable">' );<br />
  245. 245 
  246. 246 <br />
  247. 247 $this->table->set_template($tmpl);
  248. 248 </code>
  249. 249 
  250. 250 
  251. 251 <h2>$this->table->set_empty()</h2>
  252. 252 
  253. 253 <p>Let's you set a default value for use in any table cells that are empty.  You might, for example, set a non-breaking space:</p>
  254. 254 
  255. 255 <code>
  256. 256 $this->table->set_empty("&amp;nbsp;");
  257. 257 </code>
  258. 258 
  259. 259 <h2>$this->table->clear()</h2>
  260. 260 
  261. 261 <p>Lets you clear the table heading and row data.  If you need to show multiple tables with different data you should
  262. 262 to call this function after each table has been generated to empty the previous table information. Example:</p>
  263. 263 
  264. 264 <code>
  265. 265 $this->load->library('table');<br />
  266. 266 <br />
  267. 267 $this->table->set_heading('Name', 'Color', 'Size');<br />
  268. 268 $this->table->add_row('Fred', 'Blue', 'Small');<br />
  269. 269 $this->table->add_row('Mary', 'Red', 'Large');<br />
  270. 270 $this->table->add_row('John', 'Green', 'Medium');<br />
  271. 271 <br />
  272. 272 echo $this->table->generate();<br />
  273. 273 <br />
  274. 274 <kbd>$this->table->clear();</kbd><br />
  275. 275 <br />
  276. 276 $this->table->set_heading('Name', 'Day', 'Delivery');<br />
  277. 277 $this->table->add_row('Fred', 'Wednesday', 'Express');<br />
  278. 278 $this->table->add_row('Mary', 'Monday', 'Air');<br />
  279. 279 $this->table->add_row('John', 'Saturday', 'Overnight');<br />
  280. 280 <br />
  281. 281 echo $this->table->generate();
  282. 282 </code>
  283. 283 
  284. 284 <h2>$this->table->function</h2>
  285. 285 
  286. 286 <p>Allows you to specify a native PHP function or a valid function array object to be applied to all cell data.</p>
  287. 287 
  288. 288 <code>$this->load->library('table');<br />
  289. 289 <br />
  290. 290 $this->table->set_heading('Name', 'Color', 'Size');<br />
  291. 291 $this->table->add_row('Fred', '&lt;strong&gt;Blue&lt;/strong&gt;', 'Small');<br />
  292. 292 <br />
  293. 293 $this->table->function = 'htmlspecialchars';<br />
  294. 294 echo $this->table->generate();<br />
  295. 295 </code>
  296. 296 
  297. 297 <p>In the above example, all cell data would be ran through PHP's <dfn>htmlspecialchars()</dfn> function, resulting in:</p>
  298. 298 
  299. 299 <code>&lt;td&gt;Fred&lt;/td&gt;&lt;td&gt;&amp;lt;strong&amp;gt;Blue&amp;lt;/strong&amp;gt;&lt;/td&gt;&lt;td&gt;Small&lt;/td&gt;</code>
