  59.  59 <h1>Zip Encoding Class</h1>
  60.  60 <p>CodeIgniter's Zip Encoding Class classes permit you to create Zip archives. Archives can be downloaded to your
  61.  61 desktop or saved to a directory.</p>
  62.  62 
  63.  63 
  64.  64 <h2>Initializing the Class</h2>
  65.  65 <p>Like most other classes in CodeIgniter, the Zip class is initialized in your controller using the <dfn>$this->load->library</dfn> function:</p>
  66.  66 
  67.  67 <code>$this->load->library('zip');</code>
  68.  68 <p>Once loaded, the Zip library object will be available using: <dfn>$this->zip</dfn></p>
  69.  69 
  70.  70 
  71.  71 <h2>Usage Example</h2>
  72.  72 
  73.  73 <p>This example demonstrates how to compress a file, save it to a folder on your server, and download it to your desktop.</p>
  74.  74 
  75.  75 <code>
  76.  76 $name = 'mydata1.txt';<br />
  77.  77 $data = 'A Data String!';<br />
  78.  78 <br />
  79.  79 $this->zip->add_data($name, $data);<br />
  80.  80 <br />
  81.  81 // Write the zip file to a folder on your server. Name it "my_backup.zip"<br />
  82.  82 $this->zip->archive('/path/to/directory/my_backup.zip');
  83.  83 <br /><br />
  84.  84  // Download the file to your desktop.  Name it "my_backup.zip"<br />
  85.  85 $this->zip->download('my_backup.zip');
  86.  86 </code>
  87.  87 
  88.  88 <h1>Function Reference</h1>
  89.  89 
  90.  90 <h2>$this->zip->add_data()</h2>
  91.  91 
  92.  92 <p>Permits you to add data to the Zip archive. The first parameter must contain the name you would like
  93.  93 given to the file, the second parameter must contain the file data as a string:</p>
  94.  94 
  95.  95 <code>
  96.  96 $name = 'my_bio.txt';<br />
  97.  97 $data = 'I was born in an elevator...';<br />
  98.  98 <br />
  99.  99 $this->zip->add_data($name, $data);
  100. 100 </code>
  101. 101 
  102. 102 <p>You are allowed multiple calls to this function in order to
  103. 103 add several files to your archive.  Example:</p>
  104. 104 
  105. 105 <code>
  106. 106 $name = 'mydata1.txt';<br />
  107. 107 $data = 'A Data String!';<br />
  108. 108 $this->zip->add_data($name, $data);<br />
  109. 109 <br />
  110. 110 $name = 'mydata2.txt';<br />
  111. 111 $data = 'Another Data String!';<br />
  112. 112 $this->zip->add_data($name, $data);<br />
  113. 113 </code>
  114. 114 
  115. 115 <p>Or you can pass multiple files using an array:</p>
  116. 116 
  117. 117 <code>
  118. 118 $data = array(<br />
  119. 119 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'mydata1.txt' => 'A Data String!',<br />
  120. 120 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'mydata2.txt' => 'Another Data String!'<br />
  121. 121 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;);<br />
  122. 122 <br />
  123. 123 $this->zip->add_data($data);<br />
  124. 124 <br />
  125. 125 $this->zip->download('my_backup.zip');
  126. 126 </code>
  127. 127 
  128. 128 <p>If you would like your compressed data organized into sub-folders, include the path as part of the filename:</p>
  129. 129 
  130. 130 <code>
  131. 131 $name = '<kbd>personal/</kbd>my_bio.txt';<br />
  132. 132 $data = 'I was born in an elevator...';<br />
  133. 133 <br />
  134. 134 $this->zip->add_data($name, $data);
  135. 135 </code>
  136. 136 
  137. 137 <p>The above example will place <dfn>my_bio.txt</dfn> inside a folder called <kbd>personal</kbd>.</p>
  138. 138 
  139. 139 
  140. 140 <h2>$this->zip->add_dir()</h2>
  141. 141 
  142. 142 <p>Permits you to add a directory.  Usually this function is unnecessary since you can place your data into folders when
  143. 143 using <dfn>$this->zip->add_data()</dfn>, but if you would like to create an empty folder you can do so.  Example:</p>
  144. 144 
  145. 145 <code>$this->zip->add_dir('myfolder'); // Creates a folder called "myfolder"</code>
  146. 146 
  147. 147 
  148. 148 
  149. 149 <h2>$this->zip->read_file()</h2>
  150. 150 
  151. 151 <p>Permits you to compress a file that already exists somewhere on your server.  Supply a file path and the zip class will
  152. 152 read it and add it to the archive:</p>
  153. 153 
  154. 154 <code>
  155. 155 $path = '/path/to/photo.jpg';<br /><br />
  156. 156 $this->zip->read_file($path);
  157. 157 <br /><br />
  158. 158  // Download the file to your desktop.  Name it "my_backup.zip"<br />
  159. 159 $this->zip->download('my_backup.zip');
  160. 160 </code>
  161. 161 
  162. 162 <p>If you would like the Zip archive to maintain the directory structure of the file in it, pass <kbd>TRUE</kbd> (boolean) in the
  163. 163 second parameter.  Example:</p>
  164. 164 
  165. 165 
  166. 166 <code>
  167. 167 $path = '/path/to/photo.jpg';<br /><br />
  168. 168 $this->zip->read_file($path, <kbd>TRUE</kbd>);
  169. 169 <br /><br />
  170. 170  // Download the file to your desktop.  Name it "my_backup.zip"<br />
  171. 171 $this->zip->download('my_backup.zip');
  172. 172 </code>
  173. 173 
  174. 174 <p>In the above example, <dfn>photo.jpg</dfn> will be placed inside two folders:  <kbd>path/to/</kbd></p>
  175. 175 
  176. 176 
  177. 177 
  178. 178 <h2>$this->zip->read_dir()</h2>
  179. 179 
  180. 180 <p>Permits you to compress a folder (and its contents) that already exists somewhere on your server.  Supply a file path to the
  181. 181 directory and the zip class will recursively read it and recreate it as a Zip archive.  All files contained within the
  182. 182 supplied path will be encoded, as will any sub-folders contained within it.  Example:</p>
  183. 183 
  184. 184 <code>
  185. 185 $path = '/path/to/your/directory/';<br /><br />
  186. 186 $this->zip->read_dir($path);
  187. 187 <br /><br />
  188. 188  // Download the file to your desktop.  Name it "my_backup.zip"<br />
  189. 189 $this->zip->download('my_backup.zip');
  190. 190 </code>
  191. 191 
  192. 192 <p>By default the Zip archive will place all directories listed in the first parameter inside the zip. If you want the tree preceding the target folder to be ignored
  193. 193 you can pass <kbd>FALSE</kbd> (boolean) in the second parameter.  Example:</p>
  194. 194 
  195. 195 <code>
  196. 196 $path = '/path/to/your/directory/';<br /><br />
  197. 197 $this->zip->read_dir($path, FALSE);
  198. 198 </code>
  199. 199 
  200. 200 <p>This will create a ZIP with the folder "directory" inside, then all sub-folders stored correctly inside that, but will not include the folders <samp>/path/to/your</samp>.</p>
  201. 201 
  202. 202 
  203. 203 
  204. 204 
  205. 205 <h2>$this->zip->archive()</h2>
  206. 206 
  207. 207 <p>Writes the Zip-encoded file to a directory on your server.  Submit a valid server path ending in the file name.  Make sure the
  208. 208 directory is writable (666 or 777 is usually OK). Example:</p>
  209. 209 
  210. 210 <code>$this->zip->archive('/path/to/folder/myarchive.zip'); // Creates a file named myarchive.zip</code>
  211. 211 
  212. 212 
  213. 213 <h2>$this->zip->download()</h2>
  214. 214 
  215. 215 <p>Causes the Zip file to be downloaded from your server. The function must be passed the name you would like the zip file called.
  216. 216 Example:</p>
  217. 217 
  218. 218 <code>$this->zip->download('latest_stuff.zip'); // File will be named "latest_stuff.zip"</code>
  219. 219 
  220. 220 <p class="important"><strong>Note:</strong>&nbsp; Do not display any data in the controller in which you call this function since it sends various server headers
  221. 221 that cause the download to happen and the file to be treated as binary.</p>
  222. 222 
  223. 223 
  224. 224 <h2>$this->zip->get_zip()</h2>
  225. 225 
  226. 226 <p>Returns the Zip-compressed file data.  Generally you will not need this function unless you want to do something unique with the data.
  227. 227 Example:</p>
  228. 228 
  229. 229 <code>
  230. 230 $name = 'my_bio.txt';<br />
  231. 231 $data = 'I was born in an elevator...';<br />
  232. 232 <br />
  233. 233 $this->zip->add_data($name, $data);<br /><br />
  234. 234 
  235. 235 $zip_file = $this->zip->get_zip();
  236. 236 </code>
  237. 237 
  238. 238 
  239. 239 <h2>$this->zip->clear_data()</h2>
  240. 240 
  241. 241 <p>The Zip class caches your zip data so that it doesn't need to recompile the Zip archive for each function you use above.
  242. 242 If, however, you need to create multiple Zips, each with different data, you can clear the cache between calls. Example:</p>
  243. 243 
  244. 244 <code>
  245. 245 $name = 'my_bio.txt';<br />
  246. 246 $data = 'I was born in an elevator...';<br />
  247. 247 <br />
  248. 248 $this->zip->add_data($name, $data);<br />
  249. 249 $zip_file = $this->zip->get_zip();<br />
  250. 250 <br />
  251. 251 <kbd>$this->zip->clear_data();</kbd>
  252. 252 <br /><br />
  253. 253 
  254. 254 $name = 'photo.jpg';<br />
  255. 255 $this->zip->read_file("/path/to/photo.jpg"); // Read the file's contents<br />
  256. 256 <br /><br />
  257. 257 $this->zip->download('myphotos.zip');
  258. 258 </code>
